# 与 momoo 等行情看盘软件的对比

基于当前实现的自检与常见行情 App（如 momoo、东方财富、同花顺等）的对比，便于查漏补缺。

---

## 一、当前我们已具备的功能

| 功能 | 状态 | 说明 |
|------|------|------|
| **K 线 + 成交量** | ✅ | 日/周/月/年 K、蜡烛图、主图 MA/EMA、副图 VOL/MACD/RSI |
| **分时** | ✅ | 分时折线、多日（1天/2天/3天/4天）、成交量柱 |
| **左右滑动** | ✅ | 横向拖动画布，查看更早/更晚区间；长按出现十字线时拖动为移动十字线 |
| **放大缩小** | ✅ | 双指捏合缩放，可见 K 线根数 30～400 根可调 |
| **十字线 / 长按** | ✅ | 长按显示竖线 + 当日开高低收/MA/副图数值，松手或点击取消 |
| **加载更多历史** | ✅ | 向左滑到最左侧时自动请求更早数据并 prepend |
| **近 20 年补全** | ✅ | 首屏出图后后台按 800ms 间隔持续请求更早数据，直到约 20 年或接口无更多 |
| **回最新** | ✅ | 非「贴右」时显示「回最新」按钮，一键回到最新 K 线 |
| **指标栏位置** | ✅ | MA/EMA/VOL/MACD/RSI 已放在 K 线图**上方**（周期选择条下方） |
| **底部数据条** | ✅ | 开/高/低/收/昨收/涨跌/涨跌幅/振幅/均价/成交量/成交额/股息率/市盈率等，横向滚动 |
| **后端分页** | ✅ | `/api/candles` 支持 `fromMs`/`toMs`，可拉取任意时间段 |

## 二、与 momoo 等常见软件的差异（可改进点）

### 1. 数据与性能

| 对比项 | momoo 等 | 我们 | 建议 |
|--------|----------|------|------|
| **首屏数据量** | 常为近期 1～2 年，可秒开 | 默认约 2 年，依赖后端/网络 | 保持 2 年首屏，继续用后台补全到 20 年 |
| **历史深度** | 多数 10～20 年可滑到 | 设计为近 20 年，依赖接口是否给足 | 确认 Polygon/Twelve 等接口对该标的的历史范围 |
| **补全节奏** | 无感知后台拉取 | 800ms 间隔逐段拉取 | 可加「正在加载更多历史」轻提示，避免误以为卡死 |

### 2. 交互与体验

| 对比项 | momoo 等 | 我们 | 建议 |
|--------|----------|------|------|
| **滑动灵敏度** | 手指滑动与 K 线位移 1:1 或接近 | 已改为 `_panFactor = 1.0`，约 1 根宽度滑动对应 1 根 | 已对齐，可再按真机手感微调 |
| **缩放中心** | 常以双指中心为缩放中心 | 当前以整表缩放，中心未跟随手势 | 可选：缩放时以手势中心对应 K 线索引为锚点 |
| **十字线拖拽** | 十字线可左右拖拽查看不同日期 | 长按后拖动为移动十字线 | 已一致 |
| **时间/价格轴高亮** | 十字线对应位置在横轴/纵轴有高亮或小标签 | 有竖线 + 上方指标条，轴上的时间/价格高亮较弱 | 可选：在 X/Y 轴标出当前十字线日期与价格 |

### 3. 布局与信息展示

| 对比项 | momoo 等 | 我们 | 建议 |
|--------|----------|------|------|
| **指标选择** | 常在图表上方或侧边 | MA/EMA/VOL/MACD/RSI 已在 K 线图上方 | 已对齐 |
| **底部数据** | 开高低收等固定一行，可横滑 | 固定高度 + 横向滚动 | 已对齐 |
| **顶部简讯** | 部分有价/量/涨跌一句话 | 有 DetailHeader（价、涨跌、状态） | 可考虑加一句「量/额」摘要 |

### 4. 我们暂无的功能（多数 App 有）

| 功能 | 说明 |
|------|------|
| **画线/标注** | 趋势线、水平线、斐波那契等，需自绘 + 持久化 |
| **多图组合** | 同一屏多股或多周期对比 |
| **全屏/横屏图表** | 当前为页面内嵌，无单独全屏 K 线 |
| **实时推送 K 线** | 最后一根随行情更新（我们依赖定时或手动刷新） |
| **更多技术指标** | 如 BOLL、KDJ、OBV 等，需在副图/主图扩展 |

---

## 三、自检结论（与截图对照）

- **布局**：指标栏已在 K 线图上方；底部为开高低收等数据条，与常见软件一致。
- **滑动**：已支持左右拖拽移动视口；长按后拖动为移动十字线；`_panFactor` 已设为 1.0，手感更接近常见 App。
- **缩放**：双指缩放已支持，范围 30～400 根。
- **数据**：首屏约 2 年，之后后台补全至约 20 年；若截图只看到约 6 周，可能是：(1) 该标的接口只返回近期，(2) 补全尚未跑完，(3) 网络/后端限流。可看控制台 `[KLine loadMore]` / `[Backend getCandlesOlderThan]` 日志确认。
